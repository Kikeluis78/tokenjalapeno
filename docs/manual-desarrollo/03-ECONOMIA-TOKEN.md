# 🌶️ Token Jalapeño (JAL): Economía del Proyecto

## Concepto del Token

**JAL NO es para apostar, es para:**
- Coleccionar contenido cultural
- Desbloquear experiencias premium
- Participar en gobernanza comunitaria
- Recompensar engagement educativo

---

## 💎 Utilidades del Token JAL

| Función | Mecánica |
|---------|----------|
| **Coleccionar** | NFTs de cartas con arte mexicano único |
| **Desbloquear** | Cartas especiales, tableros personalizados |
| **Gobernar** | Votar próximos temas (regiones de México, épocas históricas) |
| **Stakear** | Ganar % de fees por ser "madrina/padrino" de sala |

### 1. Coleccionar NFTs
- Cada carta es un NFT coleccionable
- Diferentes raridades: Común, Raro, Épico, Legendario
- Arte único por artistas mexicanos
- Metadata con información cultural

### 2. Desbloquear Contenido
- **Cartas especiales**: Ediciones limitadas de eventos
- **Tableros personalizados**: Temas regionales (Oaxaca, Jalisco, etc.)
- **Avatares**: Personajes culturales mexicanos
- **Efectos visuales**: Animaciones premium

### 3. Gobernanza Comunitaria
**Los holders de JAL votan sobre:**
- Próxima temática de cartas (ej: Gastronomía vs Historia Prehispánica)
- Regiones de México a destacar
- Colaboraciones con instituciones culturales
- Distribución de fondos para artistas

### 4. Staking (Madrinas/Padrinos)
- Stakea JAL para "apadrinar" una sala de juego
- Ganas % de las fees de esa sala
- Incentiva liquidez y compromiso a largo plazo
- Modelo similar a "liquidity mining" pero cultural

## ⚖️ Cumplimiento Regulatorio

### ✅ Por qué JAL NO es un security:

1. **Utilidad real**: Desbloquea contenido y funcionalidades
2. **No promete ganancias**: Es para usar, no para especular
3. **Gobernanza descentralizada**: Comunidad decide el futuro
4. **Valor cultural**: Preservación de tradición mexicana

### ✅ Por qué el juego NO es azar puro:

1. **Componente educativo**: Aprendizaje de cultura
2. **Modo gratuito disponible**: No obliga a apostar
3. **Skill involucrado**: Velocidad, memoria, atención
4. **Valor social**: Experiencia familiar tradicional

## 🚨 Riesgos y Mitigación

| Riesgo | Mitigación |
|--------|------------|
| **Baja liquidez JAL** | Incentivos de staking, uso constante en juego |
| **Poca adopción** | Marketing cultural fuerte, modo escuela |
| **Regulación** | Enfoque educativo/cultural, no azar puro |
| **Competencia** | Diferenciación única: primera lotería cultural blockchain |
